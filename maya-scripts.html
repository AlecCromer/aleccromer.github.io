Three.js Hair Card Simulation Generator Tool

Using WebGL and a hair physics simulation, a user can define sliders that will create an image that can be used as a hair card.

This application allows the user to specify different regions to focus on, as well as including an AI model that can be used to determine the attributes automatically with a positive detection threshold.

Maya Python Planet Generation Tool

This script was made to generate planets using height maps. The script can take a height map, wrap it to an object, and then map the hue values from the height map to a given vertex. Add some water and you got a planet!
