Maya Scripts

Here is a small collection of some of the scripts I have made for Maya in Python

Planet generated by Planet Script. I added the blue sphere on the inside to emphasize the possibilities.

May 2019

This script was made to generate planets using height maps. The script can take a height map, wrap it to an object, and then map the hue values from the height map to a given vertex. Add some water and you got a planet!

The full code can be found here on GitHub

This script was made to place items in random locations given a range of X, Y, and Z axis. This can be used to place items such as foliage, clutter, and other various purposes. The documentation for Maya scripting is scarce, but a change that would make this better would be to allow weight painting as well as density.

The full code can be found here on GitHub
