Houdini MetaHuman Skeleton Automation Tool
This project's goal is to allow a user the ability to upload any character, and have a Houdini tool automatically determine the joints/weights needed to build the metahuman skeleton to be used with existing Unreal Engine Metahuman Animation blueprints. 

Tool's current abilities include extracting the center of major joint regions based on geometry, and systematically building finger joints based on the wrist position.

Machine Learning (AI) Rigging Study
This project explores the feasibility of generating bone structures based on provided geometry. The goal being to prove that artificial intelligence will be instrumental in automating future rigging processes. By creating systems that automatically build bone structures, we save time to allow artists to focus on creating more compelling stories!
Current developments include a l-system generator and a tube generator that are used to procedurally generate .usda files to be passed into a PyTorch dataloader and neural network.
Project is being developed with the help of a Technical Artist at Meta
